## üì• Download & Install

### **Windows**
1. Download `StatusStrap Setup X.X.X.exe` from [Latest Release](https://github.com/Orbit-Softworks/statusstrap-windows-application/releases/latest)
2. Run the installer and follow the setup wizard
3. Launch StatusStrap from Start Menu or Desktop

### **macOS**
1. Download `StatusStrap-X.X.X.zip` from [Latest Release](https://github.com/Orbit-Softworks/statusstrap-windows-application/releases/latest)
2. Double-click to extract the ZIP file
3. Drag `StatusStrap.app` to your Applications folder
4. Launch from Applications folder or Spotlight

> **Note:** On first launch, you may need to right-click and select "Open" due to security settings.

### **Linux**
1. Download `StatusStrap-X.X.X.AppImage` from [Latest Release](https://github.com/Orbit-Softworks/statusstrap-windows-application/releases/latest)
2. Make it executable:
   ```bash
   chmod +x StatusStrap-X.X.X.AppImage
3. Run the application:
   ```bash
   ./StatusStrap-X.X.X.AppImage
